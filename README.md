# FundXirr
利用XIRR算法计算基金定投年化收益率

- 目前支持: 
  - 蛋卷基金(支持组合代码[DanjuanPlan类] 和 独立基金代码[DanjuanFund类])， 比如螺丝钉组合DanjuanPlan("CSI666")
  - 睿远基金[RuiyuanFund类]

- 需要填入Cookie
